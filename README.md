# Shopping Behaviour Analysis Dashboard

## Project Overview

This project analyzes customer shopping behavior using Microsoft Excel.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Conditional Formatting

## Key Insights

* Top-selling product categories
* Customer purchase trends
* Monthly sales performance
* Preferred payment methods

## Skills Demonstrated

* Data Cleaning
* Data Analysis
* Dashboard Design
* Reporting
* Data Visualization
